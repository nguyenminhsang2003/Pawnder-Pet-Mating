# SEP490_G151 : Pawnder: Pet Dating App
# Hướng dẫn tạo database Pawnder

## Bước 1: Tạo Database
- Mở **pgAdmin**
- Chọn **Servers → PostgreSQL**
- Chuột phải chọn **Create → Database**
- Nhập tên tại ô **Database**: `pawnder_database`
- Nhấn **Save**

## Bước 2: Import file .sql
- Chọn **PostgreSQL → pawnder_database**
- Nhấn **Alt + Shift + Q** để mở Query Tool
- Nhấn **Ctrl + O** → chọn file `.sql` cần import
- Nhấn **F5** để chạy lệnh
